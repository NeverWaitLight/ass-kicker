# 模板管理系统用户指南

## 概述

模板管理系统允许用户创建、管理和使用各种类型的模板。本指南将帮助您了解如何有效地使用该系统。

## 登录系统

1. 打开浏览器并访问应用程序URL
2. 输入您的用户名和密码
3. 点击“登录”按钮

## 主界面导航

登录后，您将看到主界面，包含以下部分：
- 顶部导航栏：包含系统菜单和用户信息
- 侧边栏：快速访问不同功能模块
- 主内容区：显示当前选中的功能

## 模板管理功能

### 查看模板列表

1. 在侧边栏中点击“模板管理”
2. 您将看到所有可用模板的列表
3. 每个模板卡片显示：
   - 模板名称
   - 模板描述
   - 操作按钮（查看、编辑、删除）

### 搜索模板

1. 在模板列表页面顶部的搜索框中输入关键词
2. 系统将实时筛选匹配的模板

### 创建新模板

1. 点击“新建模板”按钮
2. 在弹出的对话框中填写以下信息：
   - **模板名称**（必填）：为模板指定一个易于识别的名称
   - **描述**（可选）：简要描述模板的用途
   - **内容**（必填）：模板的实际内容
3. 点击“创建”按钮保存模板

### 编辑模板

1. 在模板列表中找到要编辑的模板
2. 点击该模板的“编辑”按钮
3. 修改所需的字段
4. 点击“更新”按钮保存更改

### 查看模板详情

1. 在模板列表中找到要查看的模板
2. 点击该模板的“查看”按钮
3. 您将看到模板的完整详细信息

### 删除模板

1. 在模板列表中找到要删除的模板
2. 点击该模板的“删除”按钮
3. 在弹出的确认对话框中点击“确定”以确认删除
4. 注意：删除操作不可逆，请谨慎操作

## 最佳实践

### 模板命名建议

- 使用清晰、描述性的名称
- 避免使用特殊字符
- 保持一致的命名约定

### 模板内容建议

- 保持内容简洁明了
- 使用标准格式
- 定期审查和更新模板

## 常见问题解答

### Q: 我忘记了密码怎么办？
A: 点击登录页面的“忘记密码”链接，按照提示重置密码。

### Q: 如何恢复误删的模板？
A: 很抱歉，删除的模板无法恢复。请在删除前仔细确认。

### Q: 模板数量有限制吗？
A: 单个用户可以创建的模板数量没有硬性限制，但请合理使用存储空间。

### Q: 模板支持哪些格式？
A: 模板内容可以是纯文本、HTML、Markdown 或任何其他文本格式。
