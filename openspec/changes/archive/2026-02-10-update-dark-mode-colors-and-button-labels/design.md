## Context

当前系统在暗色模式下，特定背景色 #073642 的文字颜色对比度不足，导致可读性差。同时，界面上的按钮标签如"新建通道"、"新建用户"等过于冗长，占用了较多界面空间。另外，通道类型 PUSH 缺少中文名称，影响中文用户的理解。

## Goals / Non-Goals

**Goals:**
- 提高暗色模式下 #073642 背景的文字可读性
- 简化按钮标签，提升界面简洁性
- 为通道类型 PUSH 添加中文名称"系统推送"

**Non-Goals:**
- 修改其他背景色的配色方案
- 对整个系统的国际化进行全面改造
- 更改除 PUSH 类型外的其他通道类型的名称

## Decisions

1. **暗色模式文字颜色调整**：针对 #073642 背景色，使用 #faf4e8 作为文字颜色，以达到足够的对比度，满足可访问性标准。
2. **按钮标签简化**：将具有相似动作的按钮统一为相同的标签，如"新建通道"、"新建用户"都简化为"新建"，通过上下文让用户理解具体含义。
3. **国际化配置**：在现有的国际化配置文件中添加 PUSH 类型的中文名称映射。

## Risks / Trade-offs

[Risk] 文字颜色对比度可能仍然不足 -> 使用在线工具验证颜色对比度是否满足 WCAG 标准
[Mitigation] 如果 #faf4e8 对比度不够，选择其他更合适的颜色

[Risk] 简化的按钮标签可能导致用户混淆 -> 在按钮旁边保留适当的图标或提示
[Mitigation] 通过用户测试验证简化后的标签是否容易理解
